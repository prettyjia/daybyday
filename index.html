Google 캘린더 연결

Google Calendar API를 연결하려면 아래 단계를 따르세요:

  1. Google Cloud Console에서 프로젝트 생성
  2. Calendar API 활성화
  3. OAuth 2.0 클라이언트 ID 발급
  4. config.js에 클라이언트 ID 입력
// config.js
const CONFIG = {
  GOOGLE_CLIENT_ID: "your-client-id.apps.googleusercontent.com",
  BIBLE_API_KEY: "your-bible-api-key", // scripture.api.bible
  NEWS_API_KEY: "your-newsapi-key" // newsapi.org
};
